### Start All Services

.\scripts\start-dev.ps1

### Stop All Services

.\scripts\stop-dev.ps1

### Reset Database

.\scripts\reset-db.ps1

### Backup Database

.\scripts\backup-db.ps1

### Run Tests

.\scripts\run-tests.ps1

---

## Linux/Mac Bash Scripts

### Start All Services

./scripts/start-dev.sh

### Stop All Services

./scripts/stop-dev.sh

### Reset Database

./scripts/reset-db.sh

### Backup Database

./scripts/backup-db.sh

### Run Tests

./scripts/run-tests.sh

---

## Script Descriptions

**start-dev**: Starts both backend and frontend development servers
**stop-dev**: Stops all development processes
**reset-db**: Resets database and runs migrations
**backup-db**: Creates database backup
**run-tests**: Runs all test suites

---

_Note: Scripts akan dibuat sesuai kebutuhan development team_
